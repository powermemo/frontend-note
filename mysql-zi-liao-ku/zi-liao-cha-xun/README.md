# DQL資料查詢

| 子句 | 用途 |
| :--- | :--- |
| SELECT | 給定查詢的資料項目\(欄位名\) |
| FROM | 給定資料來源\(資料庫名\) |
| WHERE | 給定查詢/過濾資料條件 |
| GROUP BY | 群組 |
| HAVING | 群組篩選 |
| ORDER BY | 排序 |

## 查詢命令的子句執行順序

5:`SELECT` //哪個欄位  
1:`FROM`  //哪個表格  
2:`WHERE`   //符合條件資料  
3:`GROUP BY`  //分組  
4:`HAVING`   //分組條件  
5:`ORDER BY`  //排序
