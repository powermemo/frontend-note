# 物件

## object-fit

`object-fit:`**`cover`**`;`使圖片(或物件例如影片.畫布等)縮放至容器大小。\
🔹 寬(width)高(height)不是設在父層，寬高要設在img{}的class哦！\
舉例

```css
object-fit: cover;
```

\


備註：背景的CSS有很多，然而圖片的......🤪🤪

## object-position

`object-position:` $$x$$ `` $$y$$`;` 調整圖片(或物件例如影片.畫布等) 的位置。\
舉例

```css
object-position: center top;
```
