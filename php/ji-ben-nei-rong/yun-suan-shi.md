# 運算式

* ++ \| -- 在前，先改變變數值、再執行運算式
* ++ \| -- 在後，先執行運算式、再改變變數值

{% tabs %}
{% tab title="範例1" %}
```php
<?php
$x = 10;
$y = ++$x + 5;
echo "x : $x <br>"; //11
echo "y : $y <br>"; //16
//-----------------------
$x = 10;
$y = $x++ + 5;
echo "x : $x <br>"; //11
echo "y : $y <br>"; //15
?>
```
{% endtab %}

{% tab title="範例2" %}
```php
<?php
$a = 10;
$b = ++$a + ++$a; //a:11,12 ,(11+12)
echo "a : $a <br>"; //12
echo "b : $b <br>"; //23
//-----------------------
$a = 10;
$b = $a++ + $a++; //(10+11) a:11,12
echo "a : $a <br>"; //12
echo "b : $b <br>"; //21
?>
```
{% endtab %}
{% endtabs %}
