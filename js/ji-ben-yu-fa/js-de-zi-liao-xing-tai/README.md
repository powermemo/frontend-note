# 資料型態\(內建物件\)

## JS重要的資料型態\(7種\)：

* Number數字
* String字串
* Boolean布林
* Array陣列
* Object物件
* null空值，類整數，沒有結果。
* undefined未定義，

  例如var n →甚麼都沒給

### 也是資料型態：

* Date日期時間
* Math數學
* RegExp正規表示式

## typeof利用開發人員查詢資料型態

typeof有點像查字典，  
到瀏覽器按F12或右鍵點選「檢查」進入「開發人員工具」的「Console」頁籤查詢。
